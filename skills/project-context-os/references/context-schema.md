# Context Schema

## Required Files

- `README.md`: describe how the project uses `.context/`
- `master-plan.md`: goals, phases, milestones, acceptance criteria
- `current-status.md`: current phase, done, in progress, next, blockers
- `task-breakdown.md`: concrete task decomposition and priority
- `work-log.md`: append-only structured execution history
- `bug-log.md`: engineering problems, root causes, solutions, preventions
- `decisions.md`: architecture and product decisions
- `risk-register.md`: risks, triggers, mitigations

## Structured Entry Requirements

### work-log.md

- 时间
- 目标
- 动作
- 结果
- 验证
- 下一步

### bug-log.md

- 标题
- 现象
- 触发条件
- 影响
- 根因
- 解决方案
- 预防措施
- 状态

### current-status.md

- 当前阶段
- 已完成
- 进行中
- 下一步
- 阻塞项
