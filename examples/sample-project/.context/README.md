# Sample Project `.context` 使用说明

`.context/` 是本项目的工程操作系统。开始真实实施前先读这里，再读 `INDEX.md`、`vision.md`、`design.md`、`master-plan.md`、`current-status.md`、`task-breakdown.md` 和当前活跃 `work-log` 分片。

## 使用原则

1. 先让 `VISION + DESIGN` 通过门禁，再继续实施。
2. 只做当前这一张任务卡，范围外先记录，不自行扩张。
3. 每完成一个明确步骤，追加当前活跃 `work-log` 分片。
4. 每遇到问题，记录当前活跃 `bug-log` 分片。
5. 每做出高影响决策，使用结构化 ADR 追加到当前活跃 `decisions` 分片。
6. `INDEX.md` 由脚本维护，用来快速定位最新活跃分片。
7. 每次会话结束前，刷新 `current-status.md`。
